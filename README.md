# 农行机场贵宾厅查询系统

覆盖全球 **1,060** 个机场贵宾厅信息，支持多维度筛选查询。

## 数据概况

| 指标 | 数量 |
|------|------|
| 境内机场 | 50个 |
| 境外记录 | 1,010条 |
| 覆盖区域 | 7大洲 |
| 覆盖国家 | 132个 |
| 覆盖城市 | 528个 |

## 在线访问

- **GitHub Pages**: https://roanxu.github.io/abc-lounge-query/

## 功能特性

- **六维度筛选**: 区域、国家、城市、机场代码、出发类型、安检类型
- **三级联动**: 区域 → 国家 → 城市 联动筛选
- **境内简化展示**: 仅显示是否需要预约、携伴政策
- **境外完整展示**: 航站楼、贵宾厅名称、位置指引等详细信息

## 技术栈

- React 19 + TypeScript
- Vite
- Tailwind CSS
- Lucide React 图标

## 本地开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 数据来源

中国农业银行信用卡贵宾厅服务列表，仅供参考，以机场实际情况为准。
